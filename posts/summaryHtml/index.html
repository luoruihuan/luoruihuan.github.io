дёҖгҖҒеҲқиЎ·

    ж—¶е…үйЈһйҖқпјҢиҪ¬зңјиҝҳжңүдёӨдёӘеӨҡжңҲ19е№ҙе°ұиҝҮеҺ»дәҶпјҢжңҖиҝ‘жү“з®—жӣҙж–°дёҖзі»еҲ—иҮӘе·ұж•ҙзҗҶзҡ„е…ідәҺеүҚз«ҜзҹҘиҜҶзӮ№зҡ„дёҖдәӣжұҮжҖ»пјҢз®—жҳҜеҜ№иҮӘе·ұжҺҢжҸЎзҹҘиҜҶзҡ„дёҖдёӘжұҮжҖ»пјҢжҹҘжјҸиЎҘзјәпјҢзі»еҲ—жү“з®—еҲҶдёәеӣӣдёӘеӨ§еқ—жқҘжўізҗҶпјҢеӨ§жҰӮеҲҶдёәHTMLйғЁеҲҶгҖҒCssйғЁеҲҶ(еҢ…жӢ¬Sassз”Ёжі•)гҖҒJsйғЁеҲҶ(еҢ…жӢ¬ES6зӣёе…іеҹәзЎҖ)гҖҒжңҖеҗҺжӢ“еұ•йғЁеҲҶжҖ»з»“дёҖдәӣе…ідәҺеёёз”Ёзҡ„ж–№жі•жҲ–иҖ…жҰӮеҝөжўізҗҶгҖӮжӯӨзі»еҲ—ж–Үз« зәҜзІ№еұһдәҺжё©ж•…зҹҘж–°пјҢеӨ§йғЁеҲҶйғҪжҳҜеҹәзЎҖзҡ„зҹҘиҜҶпјҢеҰӮжңүж•ҙзҗҶзҡ„дёҚеҲ°дҪҚд№ӢеӨ„пјҢжңӣеҗ„дҪҚиҪ»жӢҚгҖӮ

дәҢгҖҒHTML

1гҖҒ HTML5 ж–°зү№жҖ§пјҢиҜӯд№үеҢ–

  • Sectionпјҡ е®ҡд№үж–ҮжЎЈдёӯзҡ„дё»дҪ“йғЁеҲҶзҡ„иҠӮгҖҒж®ө
  • Articleпјҡ е®ҡд№үжқҘиҮӘеӨ–йғЁзҡ„дёҖдёӘзӢ¬з«Ӣзҡ„гҖҒе®Ңж•ҙзҡ„еҶ…е®№еқ—пјҢдҫӢеҰӮд»Җд№Ҳи®әеқӣзҡ„ж–Үз« 
  • Headerпјҡ е®ҡд№үж–ҮжЎЈгҖҒйЎөйқўзҡ„йЎөзңүгҖӮйҖҡеёёжҳҜдёҖдәӣеј•еҜје’ҢеҜјиҲӘдҝЎжҒҜ
  • Footerпјҡ е®ҡд№үдәҶж–ҮжЎЈгҖҒйЎөйқўзҡ„йЎөи„ҡ
  • Navпјҡ е®ҡд№үдәҶдёҖдёӘй“ҫжҺҘз»„жҲҗзҡ„еҜјиҲӘйғЁеҲҶ
  • Progressпјҡ иҝӣеәҰжқЎпјҲvalueпјҢmaxпјү
  • SummaryпјҢdetails е®ҡд№үе…ғзҙ пјҢз”ЁжҲ·еҸҜд»ҘзӮ№еҮ»жҹҘзңӢжҲ–иҖ…йҡҗи—Ҹ:
  • Datalist: е®ҡд№үеҸҜйҖүж•°жҚ®зҡ„еҲ—иЎЁпјҢдёҺ input й…ҚеҗҲдҪҝз”Ё

2гҖҒжөҸи§ҲеҷЁзҡ„ж ҮеҮҶжЁЎејҸе’ҢжҖӘејӮжЁЎејҸпјҲquirks modeпјү

    з”ұжқҘпјҡз”ұдәҺеҺҶеҸІзҡ„еҺҹеӣ пјҢеңЁW3Cж ҮеҮҶеҮәеҸ°еүҚдёҚеҗҢжөҸи§ҲеҷЁеҜ№йЎөйқўзҡ„жёІжҹ“жҳҜдёҚеҗҢзҡ„гҖӮ
    еҢәеҲ«пјҡ

  • aгҖҒдёҘж јжЁЎејҸпјҢwidth жҳҜеҶ…е®№е®ҪеәҰпјҢе…ғзҙ зңҹжӯЈе®ҪеәҰ = margin + border + padding + width;жҖӘејӮжЁЎејҸпјҢwidth жҳҜе…ғзҙ е®һйҷ…е®ҪеәҰпјҢеҶ…е®№е®ҪеәҰ = width - padding - borderгҖӮ
  • bгҖҒж ҮеҮҶжЁЎејҸдёӢз»ҷspanзӯүиЎҢеҶ…е…ғзҙ и®ҫзҪ®widthе’ҢheightдёҚдјҡз”ҹж•ҲпјҢиҖҢеңЁжҖӘејӮжЁЎејҸдёӢдјҡз”ҹж•ҲгҖӮ
  • cгҖҒеңЁж ҮеҮҶжЁЎејҸдёӢпјҢдёҖдёӘе…ғзҙ зҡ„й«ҳеәҰжҳҜз”ұе…¶еҢ…еҗ«зҡ„еҶ…е®№еҶіе®ҡзҡ„пјҢеҰӮжһңзҲ¶е…ғзҙ жІЎжңүи®ҫзҪ®й«ҳеәҰпјҢеӯҗе…ғзҙ и®ҫзҪ®дёҖдёӘзҷҫеҲҶжҜ”зҡ„й«ҳеәҰжҳҜж— ж•Ҳзҡ„гҖӮ
  • dгҖҒжҖӘејӮжЁЎејҸдёӢmarginпјҡ0 autoпјҢж°ҙе№іеұ…дёӯеӨұж•ҲпјҢйңҖз”Ёtext-alignпјҡcenterгҖӮ

5гҖҒmetaж Үзӯҫ

6гҖҒCanvas

     <canvas> ж Үзӯҫе®ҡд№үеӣҫеҪўпјҢжҜ”еҰӮеӣҫиЎЁе’Ңе…¶д»–еӣҫеғҸпјҢдҪҶеҝ…йЎ»дҪҝз”Ёи„ҡжң¬пјҲjsпјүжқҘз»ҳеҲ¶еӣҫеҪўгҖӮйҰ–е…ҲйңҖиҰҒиҺ·еҸ–canvasеҜ№иұЎпјҢеҲӣе»әcontextеҜ№иұЎпјҢgetContext("2d") еҜ№иұЎжҳҜеҶ…е»әзҡ„ HTML5 еҜ№иұЎпјҢжӢҘжңүеӨҡз§Қз»ҳеҲ¶и·Ҝеҫ„гҖҒзҹ©еҪўгҖҒеңҶеҪўгҖҒеӯ—з¬Ұд»ҘеҸҠж·»еҠ еӣҫеғҸзҡ„ж–№жі•.дёӢйқўжҳҜcanvasзҡ„APIз®ҖеҚ•д»Ӣз»Қпјҡ
     aгҖҒCanvasеқҗж Ү: (canvasжҳҜдёҖдёӘдәҢз»ҙзҪ‘ж јпјҢе·ҰдёҠи§’еқҗж Үдёә0пјҢ0)пјҡ
        FillRectпјҲx,y,width,heightпјүпјҡ еңЁз”»еёғдёҠз»ҳеҲ¶width*heightзҡ„зҹ©еҪўпјҢд»Һе·ҰдёҠи§’пјҲxпјҢyпјүејҖе§ӢгҖӮ
        FillStyleпјҡ еҸҜд»ҘжҳҜCSSйўңиүІпјҢжёҗеҸҳпјҢеӣҫжЎҲй»ҳи®ӨжҳҜ#000пјҲй»‘иүІпјүгҖӮ
     bгҖҒCanvasи·Ҝеҫ„: (з»ҳеҲ¶зәҝжқЎеҝ…йЎ»дҪҝз”Ёinkж–№жі•дҫӢеҰӮstroke()пјҢfill() )

        lineTo(x,y)пјҡ иө·зӮ№еқҗж ҮпјҢеҚҠеҫ„пјҢејҖе§ӢдҪҚзҪ®пјҢз»“жқҹдҪҚзҪ®

        Fontпјҡе®ҡд№үеӯ—дҪ“
        fillTextпјҡпјҲtextпјҢxпјҢyпјүеңЁcanvasдёҠз»ҳеҲ¶е®һеҝғзҡ„ж–Үжң¬пјҲж–Үеӯ—пјҢж–Үеӯ—иө·зӮ№дҪҚзҪ®пјү
        strokeTextпјҡпјҲtextпјҢxпјҢyпјүеңЁcanvasдёҠз»ҳеҲ¶з©әеҝғзҡ„ж–Үжң¬

        addColorStopпјҡж–№жі•жҢҮе®ҡйўңиүІеҒңжӯўпјҢеҸӮж•°дҪҝз”Ёеқҗж ҮжқҘжҸҸиҝ°пјҢеҸҜд»ҘжҳҜ0иҮі1.

8гҖҒIE6/7/8 bugпјҢеҸҠе…је®№еҶҷжі•

  • 1гҖҒIE6 cssйҖүжӢ©еҷЁдёҚж”ҜжҢҒеӯҗйҖүжӢ©еҷЁ
  • 2гҖҒеңЁIEжөҸи§ҲеҷЁдёӯпјҢе®ҡдҪҚе…ғзҙ зҡ„z-indexеұӮзә§жҳҜзӣёеҜ№дәҺеҗ„иҮӘзҡ„зҲ¶зә§е®№еҷЁпјҢжүҖд»ҘдјҡеҜјиҮҙz-indexеҮәзҺ°й”ҷиҜҜзҡ„иЎЁзҺ°гҖӮи§ЈеҶіж–№жі•пјҡе…¶зҲ¶зә§е…ғзҙ е®ҡд№үz-indexпјҢжңүдәӣжғ…еҶөдёӢиҝҳйңҖиҰҒе®ҡд№үposition:relativeпјӣ

9гҖҒcss jsж”ҫзҪ®дҪҚзҪ®е’ҢеҺҹеӣ 

    jsжҳҜйҳ»еЎһеҠ иҪҪпјҢдјҡеҪұе“ҚйЎөйқўеҠ иҪҪзҡ„йҖҹеәҰпјҢеҰӮжһңjsж–Үд»¶жҜ”иҫғеӨ§пјҢз®—жі•д№ҹжҜ”иҫғеӨҚжқӮзҡ„иҜқпјҢеҪұе“ҚжӣҙеӨ§гҖӮCSSж”ҫеңЁеүҚз«ҜжҳҜйЎөйқўжёІжҹ“ж—¶йҰ–е…ҲжҳҜж №жҚ®DOMз»“жһ„з”ҹжҲҗдёҖдёӘDOMж ‘з„¶еҗҺеҠ дёҠCSSж ·ејҸз”ҹжҲҗдёҖдёӘжёІжҹ“ж ‘пјҢеҰӮжһңCSSж”ҫеңЁеҗҺйқўеҸҜиғҪйЎөйқўдјҡеҮәзҺ°й—Әи·ізҡ„ж„ҹи§үпјҢжҲ–иҖ…жҳҜзҷҪеұҸжҲ–иҖ…еёғеұҖж··д№ұж ·ејҸеҫҲдё‘зӣҙеҲ°CSSеҠ иҪҪе®ҢжҲҗгҖӮ

10гҖҒд»Җд№ҲжҳҜжёҗиҝӣејҸжёІжҹ“

    д»ҘеҫҖеҹәдәҺеӯ—з¬ҰдёІжӢјжҺҘзҡ„жЁЎжқҝеј•ж“ҺжҠҖжңҜ, жҖ§иғҪеҸҜд»Ҙ,еүҚз«ҜдёҚж–№дҫҝ,иҖҢзҺ°еңЁеҹәдәҺ Virtual DOM е®һзҺ°зҡ„еҗҺз«ҜжёІжҹ“, еүҚз«ҜиҲ’жңҚ, еҗҺз«ҜеҚҙеҸҳж…ў,еҺҹеӣ жҳҜ Virtual DOM йҡҫд»ҘеҒҡйқҷжҖҒеҲҶжһҗиҝӣиЎҢйў„зј–иҜ‘, жңҖз»Ҳйҡҫд»ҘжҸҗй«ҳжҖ§иғҪ.еҸҰеӨ–з»„д»¶зә§еҲ« Caching(зј“еӯҳ) ж–№жЎҲд№ҹдёҚеӨҹжҲҗзҶҹ, жҸҗй«ҳжҖ§иғҪзҹӯжңҹеҒҡдёҚеҲ°, йӮЈд№ҲиҖғиҷ‘еҜ№жңҚеҠЎз«ҜжёІжҹ“зҡ„е·ҘдҪңиҝӣиЎҢзј©еҮҸ.жҜ”еҰӮиҜҙ, йҰ–еұҸжёІжҹ“еӨҡе°‘еҶ…е®№? ж•ҙйЎөжёІжҹ“жҖ§иғҪдҪҺ, иғҪдёҚиғҪеҸӘжёІжҹ“йғЁеҲҶ,жҜ”еҰӮиҜҙеҸӘжёІжҹ“з¬¬дёҖеұҸдё»дҪ“еҶ…е®№, иҖҢдёӢж–№жҲ–иҖ…жӣҙиҜҰз»Ҷж•°жҚ®еңЁе®ўжҲ·з«ҜжҠ“еҸ–.е°ұжҳҜиҜҙ, жңҚеҠЎз«ҜжёІжҹ“дёҖйғЁеҲҶ, е®ўжҲ·з«ҜеҠ иҪҪдёҖйғЁеҲҶ, д»ҺиҖҢеҒҡеҮәж•ҲжһңгҖӮ

11гҖҒhtmlжЁЎжқҝиҜӯиЁҖ

    htmlжңүжү“ејҖж•ҲзҺҮй«ҳзҡ„е…ҲеӨ©дјҳеҠҝпјҢдҪҶд№ҹжңүдёҖдёӘе…ҲеӨ©зјәйҷ·вҖңдёҚж”ҜжҢҒеҠЁжҖҒиҜӯиЁҖвҖқпјҢиҝҷд№ҹжҳҜhtmlжЁЎжқҝиҜӯиЁҖеҮәзҺ°зҡ„еҺҹеӣ пјҢи®©зҪ‘з«ҷеҚідә«еҸ—htmlй«ҳж•ҲпјҢеҸҲдә«еҸ—еҶ…е®№зҡ„еҠЁжҖҒеҢ–пјӣдҪҝз”ЁжҜ”иҫғеӨҡзҡ„жЁЎжқҝеј•ж“Һд№ҹе°ұжҳҜHandlebars.jsпјҢEtplпјҢеҹәжң¬еҺҹзҗҶд№ҹе°ұжҳҜе°Ҷjsonж•°жҚ®йҖҡиҝҮдёҖдәӣж–№жі•пјҢдҫӢеҰӮз”ЁеҜ№еә”зҡ„еӯ—з¬ҰдёІеҢ№й…ҚжӣҝжҚўжү©еұ•ж–№жі•пјҲжӯЈеҲҷеҢ№й…ҚпјүпјҢиҪ¬жҚўжҲҗhtmlжЁЎжқҝзүҮж®өпјҢз„¶еҗҺдёҖдёӘе®һйҷ…зҡ„javascriptеҫӘзҺҜгҖӮ
